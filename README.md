# 📚 Smart Library Management System

A secure and user-friendly Library Management System built using Streamlit, SQLAlchemy, and SQLite. This project allows administrators to manage books, issue and return books, and monitor inventory in real-time.

---

## 🚀 Live Demo

🔗 [Click Here to Use the App](https://your-app-link.streamlit.app)

---

## 🎯 Features

- 🔐 Admin Login Authentication
- 📖 Add New Books
- 📚 Issue Books to Students
- 🔄 Return Books
- 📊 Real-time Dashboard with Metrics
- 📦 Automatic Inventory Updates
- 💾 SQLite Database Integration
- 🌍 Deployed on Streamlit Cloud

---

## 🛠️ Tech Stack

- **Frontend & Backend:** Streamlit
- **Database:** SQLite
- **ORM / Database Handling:** SQLAlchemy
- **Language:** Python
